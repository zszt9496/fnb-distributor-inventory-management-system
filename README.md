# smart-inventory-management-system

NUS Project for TIC2601 Database and Web Applications

`backend` folder stores ExpressJS backend code (API endpoints) and database configuration
`frontend` folder stores ReactJS code

To start backend:

- Have "DB Browser for SQLite" installed.
- Open a terminal `cd backend` and run `npm install` then `npm run dev`:
  - This starts backend application and sequelize ORM will create the database with tables.
  - Also populates data into the database tables.

To start backend:

- Open a terminal `cd backend` and run `npm install` then `npm start`:
  - This starts the React frontend application.
